## Descripción
Esta aplicación captura, almacena y visualiza coordenadas GNSS en tiempo real desde dos fuentes simultáneas 
(Google FLP y el chip de hardware de la tablet). Además, cuenta con una pantalla de perfil que implementa un "Modo Oscuro" persistente utilizando DataStore.
